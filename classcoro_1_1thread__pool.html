template<std::invocable< bool > Fn>
auto coro::thread_pool::set_unblock_callback ( Fn &&  fn)
inline

install unblock function

Unblock function is a custom function which is called, when thread pool needs to unblock a managed thread, which is waiting on blocking operation (for example a scheduler, which sleeps until time point is reached).

Parameters
fnfunction which is called to unblock current thread.
Returns
an object (no-copy, no-move) which must be held while unblock function is active. To uninstall the funciton, destroy returned object
